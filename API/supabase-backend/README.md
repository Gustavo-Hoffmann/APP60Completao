# Supabase Backend

Este diretório concentra o backend da aplicação baseado em Supabase.

## Conteúdo
- SQL de estrutura e RLS
- Edge Functions
- documentação de papéis e arquitetura

## Fluxo
- Auth via Supabase
- profiles define papel e hierarquia
- participants guarda participantes
- RLS controla acesso
- Edge Function create-user cria usuários de forma segura
